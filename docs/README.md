# Tài Liệu Hướng Dẫn Sử Dụng (Documentation)

Chào mừng bạn đến với tài liệu hướng dẫn sử dụng của thư viện `vn-nlp`. Thư mục này chứa các tài liệu chi tiết giúp bạn hiểu rõ và tích hợp dễ dàng thư viện vào dự án của mình.

## Danh Sách Mục Lục

1. [Giới Thiệu Chung](01-introduction.md) - Tổng quan về tính năng và cài đặt cơ bản.
2. [Kiến Trúc & Thiết Kế](02-architecture.md) - Phân tích cấu trúc thư viện và hiệu năng.
3. [Hướng Dẫn: Tokenization](03-usage-tokenization.md) - Cách chia tách từ (Word Segmentation).
4. [Hướng Dẫn: Normalization](04-usage-normalization.md) - Cách chuẩn hóa chuỗi và dấu câu tiếng Việt.
5. [Hướng Dẫn: Segmentation](05-usage-segmentation.md) - Phân tách câu (Sentence boundaries).

> **Lưu ý:** Thư viện này có thể chạy trên môi trường `no_std` (với `alloc`) và hoàn toàn zero-copy. Mọi đóng góp xin hãy xem tại [CONTRIBUTING.md](../CONTRIBUTING.md).
